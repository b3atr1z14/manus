## 🚀 Como Publicar no GitHub Pages (Permanente e Gratuito)

### Passo 1: Criar uma Conta no GitHub
1. Acesse [github.com](https://github.com)
2. Clique em **"Sign up"** (Inscrever-se)
3. Preencha seus dados (email, senha, nome de usuário)
4. Confirme seu email

### Passo 2: Criar um Novo Repositório
1. Após fazer login, clique no ícone **"+"** no canto superior direito
2. Selecione **"New repository"** (Novo repositório)
3. **Nome do repositório:** `seu-usuario.github.io`
   - Substitua `seu-usuario` pelo seu nome de usuário do GitHub
   - **Exemplo:** `beatriz.github.io`
4. Deixe marcado: **"Public"** (Público)
5. Clique em **"Create repository"** (Criar repositório)

### Passo 3: Fazer Upload dos Arquivos
1. Na página do repositório, clique em **"Add file"** → **"Upload files"**
2. Arraste e solte os seguintes arquivos:
   - `index.html`
   - `style.css`
   - Pasta `imagens/` (com todas as imagens)
3. Clique em **"Commit changes"** (Confirmar alterações)

### Passo 4: Ativar GitHub Pages
1. Vá para **"Settings"** (Configurações) do repositório
2. No menu lateral, clique em **"Pages"**
3. Em **"Source"** (Fonte), selecione a branch **"main"**
4. Clique em **"Save"** (Salvar)
5. Aguarde alguns minutos...

### Passo 5: Acessar Seu Site
Seu site estará disponível em:
```
https://seu-usuario.github.io
```

**Exemplo:** Se seu usuário é `beatriz`, o link será:
```
https://beatriz.github.io
```

## 🎨 Personalizações Futuras

### Adicionar Mais Imagens
1. Coloque a nova imagem na pasta `imagens/`
2. Abra `index.html` em um editor de texto
3. Encontre o card que deseja atualizar
4. Altere o caminho da imagem em `src="imagens/nome-da-imagem.jpg"`

### Editar Descrições
1. Abra `index.html` em um editor de texto
2. Procure pela seção do card
3. Modifique o texto dentro de `<p class="card-description">...</p>`

### Mudar Cores
1. Abra `style.css` em um editor de texto
2. Procure pela seção `:root` no topo do arquivo
3. Altere os valores das variáveis de cor:
   ```css
   --cor-destaque: #00d4ff;  /* Azul neon */
   --cor-prata: #c0c0c0;     /* Prata */
   ```

### Atualizar Informações de Contato
1. Abra `index.html` em um editor de texto
2. Procure pela seção `<!-- ========== SEÇÃO DE CONTATO ========== -->`
3. Modifique os links:
   - Email: `mailto:seu-email@example.com`
   - LinkedIn: `https://linkedin.com/in/seu-perfil`
   - WhatsApp: `https://wa.me/seu-numero`
